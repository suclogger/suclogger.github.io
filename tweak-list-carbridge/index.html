【插件推荐】Carbridge：在车载大屏上使用任意iOS应用

Posted on 2019-12-21 In jailbreak Disqus:

在CarPlay上运行任意iOS应用。

风险声明：在驾驶过程中分心是极度危险的行为，请勿通过任何方式在驾驶过程中使用手机、车载系统等电子设施，这是对乘客和其他道路上的交通参与者极度不负责任的行为。

插件名：Carbridge

源地址：packix

插件地址：https://repo.packix.com/package/com.leftyfl1p.carplay/

支持版本：iOS 11, 12 (A12/arm64e)

插件作者：leftyfl1p

关于CarPlay

iOS系统内嵌了CarPlay，旨在令用户通过汽车制造商之原生车载系统来使用、操控iOS设备并发挥其功能。目前绝大多数车载系统也支持这一协议。

2013年最初推出时，只支持官方的内置若干应用，如通话，信息和地图。

2017年开放了第三方应用接入。

carplay官方介绍

wiki

遗憾的是，CarPlay支持的主角始终是两类应用，导航和音频，而且需要严格的官方授权。

Important

Some features of this class are specific to CarPlay, which requires a special entitlement issued by Apple. Apps without the correct entitlement won’t appear on the CarPlay home screen. See http://www.apple.com/ios/carplay/ for more information.

Carbridge

在CarPlay上运行任意iOS应用的想法由来已久，前身是：ngxplay，CarPlay在运行性能，交互和迭代速度上都是十分优秀的继任者。

通过Packix源安装后，在设置菜单中找到CarBridge并选择需要开启的应用，你就可以在车载大屏上看到对应的应用。

通过CarBridge来使用iOS应用，常规的用法有通过Safari浏览网站，通过Youtube（或其他视频应用如Bilibili，一席等）播放视频，选择自己喜好的导航软件等。当然还有网友们脑洞大开的用法。
